### 页面正文
The Hardware Security Group’s (HSG) security research focuses on the development and management of foundational building-block security mechanisms and techniques that can be integrated into a wide variety of mission-critical U.S. information systems. The group’s work spans a spectrum from near-term hardening and improvement to the design and analysis of next-generation, leapahead security capabilities. Computer security depends fundamentally on the level of trust that can be established for computer software and systems. This work, therefore, focuses strongly on assurance building activities ranging from the analysis of software configuration settings to advanced trust architectures to testing tools that surface flaws in software modules.
