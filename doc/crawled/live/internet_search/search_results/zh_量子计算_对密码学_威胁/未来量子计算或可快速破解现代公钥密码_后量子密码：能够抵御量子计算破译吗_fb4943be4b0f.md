# 未来量子计算或可快速破解现代公钥密码 后量子密码：能够抵御量子计算破译吗

### 来源信息

- **页面抓取**: 成功

### 搜索摘要
August 7, 2023 - 量子计算机的快速发展有可能对现代公钥密码学形成挑战。“由于量子计算机能指数或多项式量级地加快某些复杂计算问题的求解速度，因此现代公钥密码学很有可能被量子计算技术彻底颠覆。”赵于康告诉记者，以Shor量子算法为例，其可以在多项式时间内解决大整数分解和离散对数求解等复杂数学问题，因此可以快速破解广泛使用的RSA、ECC、ElGamal等公钥密码。
